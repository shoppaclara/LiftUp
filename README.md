# LiftUp
A desktop workout application to create workout plans and execute them in live time using Tkinter in Python.

Utlizes Tkinter and .JSON files as the development framework. Tkinter is the default GUI Toolkit for Python. .JSON files are logged and stored externally in the dedicated folder to store workout lists, logs, and to allow for stastic tracking capabilities. 

LiftUp allows for users to create an unlimited number of workouts, each containing custom exercises and sets. During a workout session, the application guides the user thrugh the exercises set-by-set while tracking the number of completed sets, remaining sets, and user-enterd performance data such as reps and weight. All created sets and completed sets are saved into a structured log that can be analyzed within the application, providing a clear and interactive training experiences similar to commercial fitness apps, but without the restrictions to number of custom exercises. Here, every exercise is custom. 

# Required Python Libraries:
This program relies primarly on libraries included within the default Python 3.x installation. Only external library that must be installed is PIL. 
- Tkinter
- json
- os
- datetime
- PIL (Pillow)
